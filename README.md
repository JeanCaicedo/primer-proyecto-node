# primer-proyecto-node

Primer proyecto de Node.js: hola mundo aprendiendo a inicializar un `package.json` y a definir npm scripts personalizados.

## Contexto

Ejercicio de autoestudio (mayo 2023) para dar los primeros pasos con Node.js: crear un proyecto con `npm init`, entender `package.json` y ejecutar archivos con scripts como `start` y `saludar`. Sin dependencias externas.

## Tecnologías

- Node.js
- npm

## Ejecución

```bash
npm install
npm start            # ejecuta index.js
npm run saludar      # ejecuta hola_mundo.js
```
